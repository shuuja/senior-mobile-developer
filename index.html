AI-assisted engineering

I build with AI agents, every day

Autonomous coding agents changed how I work, and I've rebuilt my whole workflow around them. Not as a shortcut, but as a force multiplier with real engineering discipline behind it.

  • Agent-driven development. I run tools like Claude Code and Cursor as part of daily work, from planning and scaffolding through to the pull request.
  • Spec first. I write clear specs and constraints so agents produce code that fits the architecture, then I review every line before it ships.
  • Custom tooling. MCP integrations, automated code review, test generation, and CI pipelines that keep quality high at speed.
  • The payoff. I ship in days what used to take weeks, without giving up code quality or maintainability.
